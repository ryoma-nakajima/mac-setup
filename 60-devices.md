# 60. 外部デバイス / ドライバ

| 対象 | インストール |
| --- | --- |
| Silicon Labs VCP（USB-UART） | `brew install --cask silicon-labs-vcp-driver` |
| WCH CH340 / CH341 | `brew install --cask wch-ch34x-usb-serial-driver` |
| ESP32 / Arduino | `brew install arduino-cli platformio`（IDE 不要） |

いずれも System Extension の承認が必要な場合は「システム設定 → プライバシーとセキュリティ」で許可＋再起動。
